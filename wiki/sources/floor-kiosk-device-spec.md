## Key Takeaways
Recommended hardware specifications for the WorkInSync Floor Kiosk tablet device:

| Parameter | Specification |
|-----------|--------------|
| Screen Size | 25" or more |
| Aspect Ratio | 16:9 |
| OS | Android 12.0 or higher |
| Chipset | Qualcomm Quad-core / ARM Cortex |
| RAM | 12 GB+ |
| ROM | 128 GB |
| GPU | Adreno 619, 642, 650, or 730 (latest Adreno) |
| Touch | Capacitive, 10-point multi-touch |
| Bluetooth | BT 5 (2.4 GHz, 0–10 m range) |
| Wi-Fi | 802.11 b/g/n/a/ac/ax (Wi-Fi 6), 2.4 GHz + 5 GHz |
| USB | USB 3.0 × 1, USB-C × 1 (full-featured, no charging) |
| Audio | 2-track, 2 × 5W speaker |
| LAN | Supported |
| Kiosk/Lock Mode | Available — disables back button, restricts URL exit, locks user in guided mode |
| Patch Updates | Configurable — supports patch without forced restart |
